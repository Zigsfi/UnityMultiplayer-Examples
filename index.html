Unity Multiplayer Examples+Lab

For this lab, you will need to add some features to the Unity multiplayer examples I've included.

  1. Fork the project directory on github.
  2. Clone the project locally.
  3. Open the project in Unity.

Part 1: SplitScreen Keyboard Controls

  1. Open the file "scene1.unity"
  2. Click the play button. Use the WASD keys to move one cube, and IJKL to move the other.
  3. Return to edit mode by pressing the play button again.
  4. Open the input preferences under Edit > Project Settings > Input. Observe the Input fields Horizontal1, Horizontal2, Vertical1, and Vertical2
  5. Open the file MPControl.js You should see how the input fields relate to the in-game controls.
  6. Modify the input preferences and the script MPControl.js so that both players can teleport to a random position. Player 1 should do this by pressing the x key. Player 2 should do this by pressing the ',' key.
  7. Modify the input preferences and the script MPControl.js so that both players teleport to their starting positions when the spacebar is pressed.
  8. Modify the input preferences, the script MPControl.js, and the GameObjects in the world so that a third player can play using the TFGH keys and the B key

Submitting

  • Push your changes to github.
  • Submit a pull request to the original repository with your changes.
